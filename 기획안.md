# Scanly - QR 디지털 명함 서비스 기획안

## 1. 서비스 개요

**슬로건**: "한 번 스캔으로 명함 교환부터 네트워킹까지"

전통적인 종이 명함의 불편함을 해결하고, QR 코드 하나로 프로필 관리부터 네트워킹까지 원스톱으로 제공하는 디지털 명함 플랫폼

### 1.1 문제 정의
- 종이 명함 분실 및 관리 어려움
- 명함 정보 변경 시 재인쇄 비용 발생
- 받은 명함 디지털화 번거로움
- 명함 교환 후 후속 연락 누락

### 1.2 솔루션
- QR 코드 기반 즉시 프로필 공유
- 실시간 프로필 정보 업데이트
- 양방향 명함 교환 기능

| 회원 | 전체 기능 사용 | 프로필 생성, 명함 저장, 스캔 기록 등 |

## 3. 핵심 기능

### 3.1 프로필 생성 및 관리

#### 기본 정보 (필수)
- 이름, 직함, 회사명
- 연락처 (전화, 이메일)


#### 확장 정보 (선택)
- 자기소개
- 포트폴리오 링크
- 회사 위치
- 소셜 미디어 링크
- 프로필 사진

---

### 3.2 QR 코드 관리

#### QR 코드 생성
- 프로필 기반 자동 생성
- 고유 URL 부여 (scanly.io/u/{username})

#### QR 코드 공유
- 이미지 저장
- SNS 공유

---

### 3.3 QR 스캔

#### 스캔 방식
- 인앱 카메라 스캔
- 기본 카메라 앱 연동 (딥링크)

#### 스캔 결과 화면
```
┌─────────────────────────────┐
│     [프로필 사진]            │
│                             │
│     홍길동                   │
│     Product Manager         │
│     ABC Company             │
│                             │
│  📞 010-1234-5678           │
│  ✉️  hong@abc.com            │
│  🔗 linkedin.com/in/hong    │
│                             │
│         ┌─────────┐         │
│         │   저장   │         │
│         └─────────┘         │
└─────────────────────────────┘
```

---

### 3.4 명함 저장 (회원 전용)

#### 저장 플로우
```
[저장 버튼 클릭]
       │
       ▼
┌─────────────────────────────┐
│  명함이 저장되었습니다!      │
│                             │
│  내 명함도 전송하시겠습니까? │
│                             │
│  ┌─────────┐ ┌─────────┐   │
│  │   예    │ │ 아니오  │   │
│  └─────────┘ └─────────┘   │
└─────────────────────────────┘
       │
       ▼ (예 선택 시)
  [내 명함 전송 완료]
```

#### 저장 옵션
- 연락처 앱 동기화 (선택)
- 그룹/태그 즉시 지정
- 메모 추가

---

### 3.5 스캔 기록

#### 기록 정보
- 스캔 일시
- 스캔 위치 (선택적)
- 상대방 프로필 스냅샷

#### 기록 관리
- 날짜별/이름별 정렬
- 검색 기능
- 삭제/보관 처리

---

### 3.6 명함 그룹/태그 관리

#### 그룹 기능
- 기본 그룹: 전체, 즐겨찾기, 최근
- 커스텀 그룹 생성 (예: 거래처, 스타트업, 컨퍼런스)

#### 태그 기능
- 다중 태그 지정
- 색상 코드 지정
- 태그별 필터링

#### 관리 화면
```
┌─────────────────────────────┐
│  내 명함함                   │
│                             │
│  📁 전체 (47)               │
│  ⭐ 즐겨찾기 (12)            │
│  🕐 최근 (5)                │
│  ─────────────────          │
│  📁 거래처 (15)             │
│  📁 스타트업 (8)            │
│  📁 2024 컨퍼런스 (7)       │
│                             │
│  + 새 그룹 만들기           │
└─────────────────────────────┘
```

---

### 3.7 미팅 예약 (추후 개발)

#### 기능 개요
- 프로필에 미팅 가능 시간 표시
- 스캔 후 바로 미팅 요청
- 캘린더 연동 (Google, Apple)

#### 플로우
```
[프로필 스캔] → [미팅 요청] → [시간 선택] → [확정 알림]
```

## 5. 사용자 시나리오

### 시나리오 1: 첫 사용자 온보딩
1. 앱 설치 후 회원가입
2. 기본 프로필 정보 입력
3. QR 코드 자동 생성
4. 공유 방법 안내 (저장, SNS, 프린트)

### 시나리오 2: 네트워킹 행사에서 명함 교환
1. 상대방 QR 코드 스캔
2. 프로필 확인 후 저장
3. "내 명함도 전송" 팝업 → 예
4. 상대방에게 푸시 알림 발송
5. 양방향 명함 교환 완료

### 시나리오 3: 저장된 명함 관리
1. 명함함에서 최근 스캔 확인
2. 그룹 "2024 네트워킹"으로 이동
3. 태그 "잠재 고객" 추가
4. 메모 "다음 주 미팅 제안" 추가
